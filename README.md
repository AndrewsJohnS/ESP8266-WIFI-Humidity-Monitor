# ESP8266-WIFI-Humidity-Monitor
Wireless humidity Monitor for my 3D printer filament

### Wiring For ESP12E LOLIN 

| Pin On DHT11| Pin On ESP8266|
| ------------- |----------------|
| GND     | G |
| VCC     | 3V3 |
| SIG    | GPIO0 (D3 on LOLIN) |
